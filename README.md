# .github

This is the **organisation-wide `.github` repository** for [NHS Wales National Data Resource (NDR)](https://dhcw.nhs.wales/our-programmes/national-data-resource1/).

## Contents

| Path | Purpose |
|---|---|
| [`profile/README.md`](profile/README.md) | Public organisation profile displayed on the [National-Data-Resource GitHub page](https://github.com/National-Data-Resource) |
| [`ROADMAP.md`](ROADMAP.md) | Programme backlog and work areas for the NDR project board |
| [`.github/workflows/create-project-tasks.yml`](.github/workflows/create-project-tasks.yml) | Workflow to create labelled GitHub issues for all roadmap tasks |

The organisation profile introduces the NDR, its mission, and the **GitHub GIG Cymru** national NHS Wales GitHub service.

## Project Board

The NDR programme backlog is structured around four work areas:

| Work Area | Description |
|---|---|
| 🏛️ **Operating Model & Governance** | National consistency, governance, and operating model |
| 🚀 **Platform Enablement & Onboarding** | Platform setup, onboarding playbooks, and starter templates |
| 🔧 **Technical Delivery & Standards** | Pipeline patterns, IaC, CI/CD, data modelling, and FHIR standards |
| 📊 **Analytics & Insight** | Analytics patterns, dashboards, and visualisation standards |

See [`ROADMAP.md`](ROADMAP.md) for the full backlog, or run the [Create Project Board Tasks](.github/workflows/create-project-tasks.yml) workflow to generate issues automatically.
